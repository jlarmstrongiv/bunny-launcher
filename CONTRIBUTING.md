# Contributing

- Compare with:
  - `bunny-launcher`
  - `create-bunny-launcher`
  - `bunny-sync`
  - `bunny-move`
- `corepack enable npm`
- Update `docs/public/static/json-schema/bunny-launcher/schema.json`
- `npm login`
- `npm install`
- `npm run prepublishOnly`
- `git add -A`
- `git commit -m "message"`
- `npm version patch`
- `git push`
- `npm publish`
